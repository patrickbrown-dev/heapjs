Documentation

Installation

node.js

Use npm to install the heap.js package.

npm install heapjs

Require it like this:

var heapjs = require("heap.js");

Instance a new heap object.

var h = new heapjs.maxHeap();

Now you're ready to interact with your heap object!

h.push(7);
